==================================
How to use 2-matrix_divided module
==================================

Importing the function:
>>> matrix_divided = __import__('2-matrix_divided').matrix_divided

Normal usage:
>>> matrix = [[1, 2, 3], [4, 5, 6]]
>>> matrix_divided(matrix, 3)
[[0.33, 0.67, 1.0], [1.33, 1.67, 2.0]]

Checking original matrix (should not be modified):
>>> matrix
[[1, 2, 3], [4, 5, 6]]

Division by zero:
>>> matrix_divided(matrix, 0)
Traceback (most recent call last):
    ...
ZeroDivisionError: division by zero

Div is not a number:
>>> matrix_divided(matrix, "2")
Traceback (most recent call last):
    ...
TypeError: div must be a number

Matrix row size mismatch:
>>> matrix_mismatch = [[1, 2, 3], [4, 5]]
>>> matrix_divided(matrix_mismatch, 2)
Traceback (most recent call last):
    ...
TypeError: Each row of the matrix must have the same size

Non-integer/float in matrix:
>>> matrix_strings = [[1, 2], [3, "4"]]
>>> matrix_divided(matrix_strings, 2)
Traceback (most recent call last):
    ...
TypeError: matrix must be a matrix (list of lists) of integers/floats
